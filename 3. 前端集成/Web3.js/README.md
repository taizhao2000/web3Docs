# Web3.js

Web3.js 是以太坊生态最经典的 JavaScript 库（2015 年）。现已进入维护模式，新项目推荐使用 ethers.js 或 viem。

## 现代前端栈推荐

```
React DApp →  wagmi + viem + RainbowKit（最佳开发体验）
非 React  →  viem（轻量、Tree-shakeable）
脚本/后端  →  ethers.js v6（简单直接）
```
