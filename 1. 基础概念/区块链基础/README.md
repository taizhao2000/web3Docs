# 区块链基础

## 概述

区块链是一种分布式账本技术，通过密码学和共识机制确保数据的不可篡改性和透明性。

## 核心概念

- **区块（Block）**：存储交易数据的数据结构
- **链（Chain）**：通过哈希链接的区块序列
- **共识机制（Consensus）**：节点间达成一致的算法
- **节点（Node）**：参与网络的计算机

## 共识机制类型

| 类型 | 代表链 | 特点 |
|------|--------|------|
| PoW | Bitcoin | 安全性高，能耗大 |
| PoS | Ethereum 2.0 | 能耗低，质押参与 |
| DPoS | EOS | 委托投票，高吞吐 |
| PoA | 私有链 | 权威节点验证 |
