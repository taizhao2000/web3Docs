# 加密货币基础

## 概述

加密货币是基于密码学原理的数字资产，运行在区块链网络上，用于价值传输和激励机制。

## 核心概念

- **代币（Token）**：链上发行的数字资产
- **钱包（Wallet）**：管理私钥和签名的工具
- **交易（Transaction）**：链上状态变更的操作
- **Gas**：执行交易所需的手续费
- **Nonce**：账户交易计数器，防重放

## 代币分类

| 类型 | 标准 | 说明 |
|------|------|------|
| 原生代币 | - | 链原生资产，如 ETH、BTC |
| 同质化代币 | ERC-20 | 可互换的代币，如 USDT |
| 非同质化代币 | ERC-721 | 独一无二的代币，如 NFT |
| 多代币 | ERC-1155 | 单合约管理多种代币 |

## 钱包类型

- **热钱包**：在线钱包，方便但风险较高（MetaMask）
- **冷钱包**：离线存储，安全性高（Ledger、Trezor）
- **合约钱包**：智能合约钱包，支持多签和恢复（Gnosis Safe）
