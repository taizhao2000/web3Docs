# 智能合约

## 概述

智能合约是部署在区块链上的自执行程序，当预设条件被满足时自动执行合约条款。

## 快速导航

### 本质理解
- 自动售货机类比：投币即出货，无需信任人
- 传统合约 vs 智能合约：信任链从5步缩减为1步
- 保险理赔案例：为什么需要智能合约

### 不是什么
- 不是法律合约、不是 AI、不一定去中心化、不一定安全
- 两大"不能"：不能主动行动、不能访问外部世界
- 确定性约束：为什么这些限制是安全设计

### 运行模型
- EVM 执行流程
- 三种调用方式（call / transaction / staticcall）
- 合约间调用与 DeFi 乐高

### 代价与取舍
- 五大代价：不可修改、执行要钱、性能差、存储贵、无法升级
- 什么时候该用、什么时候不该用
