# 智能合约

## 概述

智能合约是部署在区块链上的自执行程序，当预设条件被满足时自动执行合约条款。

## 核心特性

- **自动执行**：条件满足即自动运行，无需中介
- **不可篡改**：部署后代码无法修改（需通过代理模式升级）
- **确定性**：相同输入始终产生相同输出
- **透明性**：代码和状态对所有人可见

## 生命周期

1. **编写**：使用 Solidity/Vyper 等语言
2. **编译**：编译为字节码和 ABI
3. **测试**：单元测试和形式化验证
4. **部署**：通过交易部署到链上
5. **交互**：通过交易或调用与合约交互
6. **升级**（可选）：通过代理模式实现逻辑升级

## 相关标准

- ERC-20：同质化代币标准
- ERC-721：非同质化代币标准
- ERC-1155：多代币标准
