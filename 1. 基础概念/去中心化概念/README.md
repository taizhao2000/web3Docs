# 去中心化概念

去中心化是 Web3 的核心理念，旨在消除单点控制和信任假设，将权力分散到网络参与者，实现公开、公正、不可篡改的协作模型。

## 关键概念

- **去中心化自治组织（DAO）**：通过公开的智能合约将组织规则代码化，由代币持有者共同决策治理的组织。
- **去中心化应用（DApp）**：运行在去中心化 P2P 账本网络上、前端动态托管、后端规则由智能合约见证的永续在线应用。
- **去中心化金融（DeFi）**：通过代码代替传统银行、清算所、券商等中介的去中介化金融应用生态。
- **去中心化存储**：利用密码学和存储激励证明实现文件分散多节点容灾存储的方案（如 IPFS、Filecoin、Arweave）。
- **去中心化身份（DID）**：用户不依赖任何中心化平台（如微信/谷歌账号）独立控制、用非对称私钥证明所有权的数字自主身份。

## 去中心化程度分类

| 层级 | 说明 | 示例 |
|------|------|------|
| **完全去中心化** | 无任何中心化控制，全网节点地位平等，靠数学共识维护。 | **Bitcoin / Ethereum** |
| **半去中心化 (联盟/混合)** | 部分依赖多签多组委员会决策（如 DAO 多签），部分逻辑仍有托管成分。 | **MakerDAO / Uniswap 治理** |
| **伪去中心化 (CeFi/侧链)** | 实际决策或验证人席位高度集中在少数大实体手中，存在单点控制。 | **部分联盟链 / 高度集中的侧链** |
