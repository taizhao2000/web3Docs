# 去中心化概念

## 概述

去中心化是 Web3 的核心理念，旨在消除单点控制和信任假设，将权力分散到网络参与者。

## 关键概念

- **去中心化自治组织（DAO）**：通过智能合约实现的组织治理
- **去中心化应用（DApp）**：运行在去中心化网络上的应用
- **去中心化金融（DeFi）**：无需传统金融中介的金融服务
- **去中心化存储**：IPFS、Filecoin 等分布式存储方案
- **去中心化身份（DID）**：用户自主控制的数字身份

## 去中心化程度

| 层级 | 说明 | 示例 |
|------|------|------|
| 完全去中心化 | 无任何中心化控制 | Bitcoin |
| 半去中心化 | 部分依赖中心化服务 | MakerDAO |
| 伪去中心化 | 名义上去中心化但实际控制集中 | 部分侧链 |
