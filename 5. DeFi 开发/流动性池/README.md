## 核心要点速查

- **LP Token 的本质**：它是一个 ERC20 代币，代表你在池子里占总资产的比重。
- **无常损失的成因**：无常损失是“逆向选择”的结果，套利者总是在池子里用便宜的代币换走你贵的代币。
- **$O(1)$ 算法核心**：通过维护一个全局累加器 `rewardPerTokenStored`，将奖励的发放从“主动推送”转为“被动拉取”，在质押（Stake）、提现（Withdraw）、领奖（GetReward）时进行瞬时数学结算。
