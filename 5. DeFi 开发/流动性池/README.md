# 流动性池

## 概述

流动性池是 DeFi 的基础设施，通过智能合约锁定代币对，为交易提供流动性。

## 核心概念

- **LP Token**：流动性提供者凭证
- **无常损失（Impermanent Loss）**：价格偏移导致的相对损失
- **费率**：交易手续费分配给 LP
- **TVL**：总锁仓量

## 流动性计算

### 恒定乘积公式

```
x * y = k

其中：
x = Token A 数量
y = Token B 数量
k = 恒定常数
```

### 价格影响

```
Δy = y * (1 - x/(x + Δx))

Δx = 输入的 Token A 数量
Δy = 输出的 Token B 数量
```
