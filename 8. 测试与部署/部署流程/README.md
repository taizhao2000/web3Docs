# 部署流程与成本控制

在智能合约进入主网前，评估部署的 Gas 成本、以及升级设计（UUPS / 透明代理）带来的长期维护开销是架构师的必修课。

---

## 核心学习模块

> 📘 **[智能合约部署与升级成本](./智能合约部署与升级成本.md)**
> 本模块涵盖：
> 1. **部署成本拆解**：解析 constructor 初始化写 Slot 消耗、代码量（Bytecode Size）带来的每字节部署 SSTORE 计费规则。
> 2. **升级代理成本**：UUPS 升级（极轻 Proxy 体积）与透明代理、Beacon 代理在部署和用户高频调用时产生的 `delegatecall` 损耗对比。
> 3. **部署实战要点**：通过 Hardhat Ignition 和 Forge Script 自动化控制多环境部署、Gas 限价策略以及安全广播。
